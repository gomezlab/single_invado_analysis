CytoProbe
CytoProbe is a tool to generate and analyse FRET images. 
Images can be multiplied, divided, scaled and much more. 
The operators work always on the entire image stack in 
the specified directory. 

The following operations are available
  • Image 1 * Image 2 
Can be used to mask image 1. For that Image 2 has to be a 
BW image of the same size as image 1.
  • Image 1 / Image 2
Ratio between image 1 and image 2, for example to calculate
the FRET image
  • Image 1 * Vector
The vector must contain the same number of elements as 
images 1. The vector can be obtained from:
  • Bleach coefficient vector from image 1
The coefficients are obtained from a second order polynomial
approximation of the average intensities of images 1
            vector(frame) = a frame^2 + b frame +c
  • (Image 1 -factor Image 2) / Image 2
FRET calculation for Rac1
  • Activity (Img 1) – dist. Edge plot (img 2: mask)
Extracts the activity as a function of the distance from
the cell edge. For that the activity is average across the entire
cell edge.

Cdc42 FRET:
Get the bleaching calibration curve from the ISO image.
Get the Cdc42 activity from the ISO/EGFP ratio.

Rho1 FRET:
Corrected Fret:            FRET_C = (FRET- Bleedthrough_coefficient * CFP)
Normalized Fret:         FRET_N = FRET_C / CFP
where  Bleedthrough_coefficient = 0.371
